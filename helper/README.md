# helper-jre

`helper-jre` 是 JVM 辅助工具。

## 简介

- [x] Dates：对 `Date` 类的格式化和解析。
- [x] Environments：环境变量。
- [x] Namings：根据类信息进行命名。
- [x] RandomStrings：随机生成字符串。
- [x] StackTraces：堆栈打印。
- [x] UserNames：账号检测。
